###### 언어 특화 API & SDK 구현체

**리소스 탐지기**

[리소스][otel-concepts-resources]는 레소스 속성으로 텔레메트리를 만들어내는 엔티티를 나타낸다. 예를 들어, Kubernetes 컨테이너에서 실행되는 텔레메트리를 만들어내는 프로세스는 파드 이름, 네임스페이스, 어쩌면 디플로이먼트 이름을 갖는다. 이 속성들을 모두 리소스에 포함시킬 수 있다.

OpenTelemetry의 언어 특화 구현체는 `OTEL_RESOURCE_ATTRIBUTES` 환경 변수와 프로세스 선타임, 서비스, 호스트 또는 운영 체제와 같은 많은 공용 엔티티에 대해 리소스 탐지를 지원한다.

자세한 정보는 [리소스][otel-concepts-resources]를 참고하라.

[otel-concepts-resources]: https://opentelemetry.io/docs/concepts/resources/
