###### 언어 특화 API & SDK 구현체

**제로 코드 계측**

적용 가능하면 OpenTelemetry의 언어 특화 구현은 소스 코드를 건드리지 않고 애플리케이션을 계측할 수 있는 방법을 제공한다. 기반 메커니즘은 언어에 따라 다르지만 제로 코드 계측은 OpenTelemetry PAI와 SDK 기능을 애플리케이션에 더한다. 추가로, 몇 가지 계측 라이브러리와 익스포터 의존성을 추가할 수 있다.

더 자세한 정보는 [제로 코드 계측][otel-concepts-instrumentation-zero-code]을 참고하라.

[otel-concepts-instrumentation-zero-code]: https://opentelemetry.io/docs/concepts/instrumentation/zero-code/
