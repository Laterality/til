###### 언어 특화 API & SDK 구현체

OpenTelemetry는 선택한 언어와 함께 OpenTelemetry API를 사용해 텔레메트리 데이터를 생성하고 데이터를 선호하는 백엔드로 익스포트하는 언어 SDK를 가지고 있다. 이러한 SDK들은 애플리케이션에서 수동 계측에 연결하는 데 사용할 수 있는 공용 라이브러리와 프레임워크에 계측 라이브러리를 함께 사용할 수 있도록 한다.

자세한 내용은 [계측][otel-concepts-instrumentation]을 참고하라.

[otel-concepts-instrumentation]: https://opentelemetry.io/docs/concepts/instrumentation/
