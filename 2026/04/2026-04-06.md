###### OpenTelemtry 로그

**언어 지원**

OpenTelemetry 명세에서 로그는 [안정적인][otel-versioning-stable] 신호다. 로그 API 및 SDK의 개별 언어에 특화된 구현 상태는 다음과 같다:

| 언어                              | 로그 |
| --------------------------------- | ---- |
| [C++][otel-lang-cpp]              | 안정 |
| [C#/.NET][otel-lang-dotnet]       | 안정 |
| [Erlang/Elixir][otel-lang-erlang] | 개발 |
| [Go][otel-lang-go]                | 베타 |
| [Java][otel-lang-java]            | 안정 |
| [JavaScript][otel-lang-js]        | 개발 |
| [PHP][otel-lang-php]              | 안정 |
| [Python][otel-lang-python]        | 개발 |
| [Ruby][otel-lang-ruby]            | 개발 |
| [Rust][otel-lang-rust]            | 베타 |
| [Swift][otel-lang-swift]          | 개발 |
