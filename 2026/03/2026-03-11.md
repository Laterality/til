##### 트레이스

###### 스팬

**스팬**은 작업 또는 연산 단위를 나타낸다. 스팬은 트레이스의 빌딩 블록이다. OpenTelemetry에서 이들은 다음 정보를 포함한다:

* 이름
* 부모 스팬 ID (루트 스팬인 경우 비어 있음)
* [스팬 컨텍스트][otel-concepts-signals-traces-span-context]
* [속성][otel-concepts-signals-traces-attributes]
* [스팬 이벤트][otel-concepts-signals-traces-span-events]
* [스팬 링크][otel-concepts-signals-traces-span-links]
* [스팬 상태][otel-concepts-signals-traces-span-status]

샘플 스팬:

```json
{
  "name": "/v1/sys/health",
  "context": {
    "trace_id": "7bba9f33312b3dbb8b2c2c62bb7abe2d",
    "span_id": "086e83747d0e381e"
  },
  "parent_id": "",
  "start_time": "2021-10-22 16:04:01.209458162 +0000 UTC",
  "end_time": "2021-10-22 16:04:01.209514132 +0000 UTC",
  "status_code": "STATUS_CODE_OK",
  "status_message": "",
  "attributes": {
    "net.transport": "IP.TCP",
    "net.peer.ip": "172.17.0.1",
    "net.peer.port": "51820",
    "net.host.ip": "10.177.2.152",
    "net.host.port": "26040",
    "http.method": "GET",
    "http.target": "/v1/sys/health",
    "http.server_name": "mortar-gateway",
    "http.route": "/v1/sys/health",
    "http.user_agent": "Consul Health Check",
    "http.scheme": "http",
    "http.host": "10.177.2.152:26040",
    "http.flavor": "1.1"
  },
  "events": [
    {
      "name": "",
      "message": "OK",
      "timestamp": "2021-10-22 16:04:01.209512872 +0000 UTC"
    }
  ]
}
```

스팬은 중첩될 수 있다. 부모 스팬 ID의 존재로 이를 알 수 있다. 자식 스팬은 서브 연산을 나타낸다. 이는 스팬들이 애플리케이션에서 이뤄진 작업을 더 정확하게 포착할 수 있게 한다.
