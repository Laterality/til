# 2026-02-26

## OpenTelemtry

### 개념

#### 컨텍스트 전파

##### 예시

`POST /cart/add`와 HTTP 엔드포인트 `GET /product`를 통해 다운스트림 서비스 `Product Catalog`에서 사용자가 장바구니에 추가하거나 체크아웃의 일부로 제품의 상세정보를 조회하는  `GET /checkout/` 엔드포인트를 가진  `Frontend`라는 서비스가 있다. `Frontend` 에서 오는 요청의 컨텍스트에서 `Product Catalog` 서비스의 활동을 이해하기 위해 W3C TraceContext 사양에 정의된 대로 `traceparent` 헤더를 사용해 컨텍스트(여기서는 트레이스 ID와 스팬 ID가 "부모 ID")가 전파된다. 즉, ID들이 헤더 필드에 삽입된다:

```
<version>-<trace-id>-<parent-id>-<trace-flags>
```

```
00-a0892f3577b34da6a3ce929d0e0e4736-f03067aa0ba902b7-01
```
