#### 기타 프로토콜

##### DynamoDB

Envoy는 다음 기능과 함께 HTTP 수준 DynamoDB 스니핑 필터를 지원한다:

- DynamoDB API 요청/응답 파서.
- DynamoDB 작업/테이블/파티션 별 작업 통계
- JSON으로 파싱된 4xx 응답에 대한 실패 통계, e.g., ProvisionedThroughputExceededException.
- 배치 연산 부분 실패 통계.

DynamoDB 필터는 Envoy의 확장성과 코어 추상화의 좋은 예시이며 모든 애플리케이션과 DynamoDB 데이터베이스 사이의 필터에 사용할 수 있다. 애플리케이션 플랫폼과 사용중인 특정 AWS SDK에 관계없이 귀중한 데이터 소스를 제공한다.
