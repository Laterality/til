###### 구성 속성

요청이나 커넥션 처리와 관련된 구성 식별자와 메타데이터는 다음 속성으로 사용할 수 있다:

| 속성                                | 타입                                         | 설명                                                                                            |
| ----------------------------------- | -------------------------------------------- | ----------------------------------------------------------------------------------------------- |
| xds.node                            | [Node][api-common-common-types-node]         | 로컬 노드 설명                                                                                  |
| xds.cluster_name                    | string                                       | 업스트림 클러스터 이름                                                                          |
| xds.cluster_metadata                | [Metadata][api-common-common-types-metadata] | 업스트림 클러스터 메타데이터                                                                    |
| xds.listener_direction              | int                                          | [리스너 트래픽 방향][api-listeners-listener-configuration-listener-traffic-direction]의 열거 값 |
| xds.listener_metadata               | [Metadata][api-common-common-types-metadata] | 리스너 메타데이터                                                                               |
| xds.route_name                      | string                                       | 라우트 이름                                                                                     |
| xds.route_metadata                  | [Metadata][api-common-common-types-metadata] | 라우트 메타데이터                                                                               |
| xds.virtual_host_name               | string                                       | 가상 호스트 이름                                                                                |
| xds.virtual_host_metadata           | [Metadata][api-common-common-types-metadata] | 가상 호스트 메타데이터                                                                          |
| xds.upstream_host_metadata          | [Metadata][api-common-common-types-metadata] | 업스트림 호스트 메타데이터                                                                      |
| xds.upstream_host_locality_metadata | [Metadata][api-common-common-types-metadata] | 업스트림 호스트 지역성 메타데이터                                                               |
| xds.filter_chain_name               | string                                       | 리스너 필터 체인 이름                                                                           |

[api-common-common-types-node]: https://www.envoyproxy.io/docs/envoy/latest/api-v3/config/core/v3/base.proto#envoy-v3-api-msg-config-core-v3-node
[api-common-common-types-metadata]: https://www.envoyproxy.io/docs/envoy/latest/api-v3/config/core/v3/base.proto#envoy-v3-api-msg-config-core-v3-metadata
[api-listeners-listener-configuration-listener-traffic-direction]: https://www.envoyproxy.io/docs/envoy/latest/api-v3/config/listener/v3/listener.proto#envoy-v3-api-field-config-listener-v3-listener-traffic-direction
