###### Wasm 속성

앞의 것들 외에도 다음 추가 속성들을 Wasm 확장에 사용할 수 있다:

| 속성           | 타입   | 설명             |
| -------------- | ------ | ---------------- |
| plugin_name    | string | 플러그인 이름    |
| plugin_root_id | string | 플러그인 루트 ID |
| plugin_vm_id   | string | 플러그인 VM ID   |
