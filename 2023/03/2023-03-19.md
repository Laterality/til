## Elasticsearch(7.10)

### 데이터 검색 - 검색 결과 정렬

#### 정렬 값

각 도큐먼트에 대한 정렬 값 또한 응답의 일부로 반환된다.

#### 정렬 순서

`order` 옵션은 다음 값을 가질 수 있다:

| 값     | 설명                   |
| ------ | ---------------------- |
| `asc`  | 오름차순으로 정렬한다. |
| `desc` | 내림차순으로 정렬한다. |

`_score`로 정렬할 때 순서의 기본값은 `desc`이고 나머지의 경우에는 `asc`이다.
