##### 서브 집계

다음 예제는 도큐먼트를 서로 다른 버킷으로 "버킷화"할 뿐만아니라 각 가격 범위에 대해 통계도 계산한다:

```http
GET /_search
{
  "aggs": {
    "price_ranges": {
      "range": {
        "field": "price",
        "ranges": [
          { "to": 100 },
          { "from": 100, "to": 200 },
          { "from": 200 }
        ]
      },
      "aggs": {
        "price_stats": {
          "stats": { "field": "price" }
        }
      }
    }
  }
}
```
